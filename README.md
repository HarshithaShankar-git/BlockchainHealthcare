# DECENTRALIZED HEALTH MONITORING  
### A Blockchain Approach to Patient Data Security (WeCare)

WeCare is a decentralized health monitoring system where patient medical records are securely stored on IPFS and referenced on a blockchain (Ethereum/Ganache). The system also includes role-based access panels for Admin, Doctor, and Patient with a user-friendly React interface.
